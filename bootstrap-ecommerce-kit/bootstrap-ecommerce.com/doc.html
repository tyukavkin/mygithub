Bootstrap e-commerce - UI KIT

A well sought-out and beautiful design system for all type of e-commerce projects based on latest version of Bootstrap

Bootstrap-ecommerce gives you a full solution for all your e-commerce websites, booking websites, product catalogs and landing pages. Speed up your design workflow with ready-to-use beautifully designed sample components for your daily projects.
Note: All components and elements are based on bootstrap frameworks. We have enhanced the components and added more elements for your ecommerce projects.

Benefits

The main purpose of this kit is to give a full solution to designers and developers in the categories of ecommerce, online marketplaces, booking platforms and product landing pages. This kit provides multiple combinations of examples and sample pages for you to work on including numerous components and element styles.

What this kit includes

Bootstrap ecommerce offers easy-to-use and fully customizable elements including CSS and HTML files along with the templates

  • Well organized file structure
  • Fonts and Fonticons, Demo images
  • SASS files with minimal code and variables
  • jQuery and basic script codes, slider samples
  • Clean code for HTML, CSS and SASS based on latest Bootstrap
  • 40+ ready-to-use components
  • Ready-to-use HTML templates

Download

Designer version

With the designer version of Bootstrap-ecommerce you will get a full package of interactive, reusable and customizable UI components and sample pages which are compatible for both SketchApp and Figma.

Developer version

The developer version of Bootstrap-ecommerce comes with clearly-coded and sought-out source files in HTML, CSS and SASS. Over 100+ icons which are related to ecommerce are fully customizable and easy-to-use and easy-to-implement in your ecommerce projects. The icons come in SVG and Fonticon formats for flexible usability.

CSS

Copy and paste the stylesheet <link> into your <head> before all other stylesheets to load our CSS. 

		
			

			<!-- Bootstrap -->
			<link href="css/bootstrap.css" rel="stylesheet" type="text/css"/>

			<!-- Font awesome 5 -->
			<link href="fonts/fontawesome/css/all.min.css" type="text/css" rel="stylesheet">

			<!-- ui style -->
			<link href="css/ui.css" rel="stylesheet" type="text/css"/>
			<link href="css/responsive.css" rel="stylesheet" />

JS

Most of bootstrap components require the use of JavaScript to function. Specifically, they require jQuery, Bootstrap.bundle.js.
We added some popular plugins in the /plugins folder like owlslider and slickslider. Our script.js file contains some additional scripts which may come handy for your project. 

		
			
			<!-- jQuery -->
			<script src="js/jquery-2.0.0.min.js" type="text/javascript"></script>

			<!-- Bootstrap -->
			<script src="js/bootstrap.bundle.min.js" type="text/javascript"></script>

			<!-- custom javascript -->
			<script src="js/script.js" type="text/javascript"></script>
